This project presents an interactive Road Accident Analysis Dashboard built using Power BI to analyze accident trends, casualties, vehicle involvement, and road conditions. 
The dashboard helps identify major factors contributing to accidents and provides meaningful insights for better decision-making.
Key Highlights:
1.Analyzed total accidents, casualties, and severity levels.
2.Compared accidents by vehicle type, road type, and weather conditions.
3.Identified monthly and yearly accident trends.
4.Examined urban vs rural accident distribution.
5.Created interactive filters and visuals for easy data exploration.
TOOLS USED : Power BI, CSV Dataset, Data Cleaning & Transformation, DAX Measures
INSIGHTS GENERATED :
1.Peak accident periods and high-risk months identified.
2.Cars and motorcycles contributed to major accidents.
3.Urban areas reported higher accident counts.
4.Weather and road surface conditions impacted accident rates.
PURPOSE : The aim of this project is to transform raw accident data into actionable insights using visualization ana analytics.
